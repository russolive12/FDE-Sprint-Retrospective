Connect your database

Paste your Supabase credentials below. This is a one-time step — after saving, anyone who opens this page will be live instantly with no extra setup.

Don't have Supabase yet? Follow the 4 steps below first — it takes about 5 minutes and is completely free. No credit card needed.
1
Go to supabase.com → click Start your project → sign in with GitHub or email.
2
Click New project. Give it any name (e.g. retro-board). Choose the region closest to your team. Set a database password and click Create new project. Wait ~1 minute for it to spin up.
3
In your project, go to the SQL Editor (left sidebar). Paste and run this SQL to create the two tables:

create table notes ( id uuid primary key default gen_random_uuid(), text text not null, theme text not null, subcat text default '', author text default '', created_at timestamptz default now() ); create table votes ( id uuid primary key default gen_random_uuid(), note_id uuid references notes(id) on delete cascade, voter_id text not null, unique(note_id, voter_id) ); alter table notes enable row level security; alter table votes enable row level security; create policy "public read notes" on notes for select using (true); create policy "public write notes" on notes for insert with check (true); create policy "public delete notes" on notes for delete using (true); create policy "public update notes" on notes for update using (true); create policy "public read votes" on votes for select using (true); create policy "public write votes" on votes for insert with check (true); create policy "public delete votes" on votes for delete using (true);
4
Go to Project Settings → API. Copy the Project URL and the anon public key. Paste them below.
